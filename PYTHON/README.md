# Backend Learning (Python)

A hands-on Python learning repository containing beginner-to-intermediate backend programming fundamentals, organized by topic with practical exercises.

## 📚 What this repository covers

This project is structured as a step-by-step learning path:

- **01_BASICS** – Variables, input, typecasting, operators, and data types
- **02_CONTROL_FLOW** – If/else, loops, logical operators, match-case, and scope
- **03_STRING_METHODS** – String operations and indexing
- **04_DATA_STRUCTURES** – Lists, sets, tuples, dictionaries, and comprehensions
- **05_FUNCTIONS** – Defining and using functions
- **06_MODULES** – Working with custom modules
- **07_OBJECT_ORIENTED_PROGRAMMING** – Classes, inheritance, polymorphism, decorators, magic methods, etc.
- **08_EXCEPTION_HANDLING** – Handling and raising exceptions
- **09_FILE_HANDLING** – Reading/writing files (TXT, CSV, JSON)
- **10_MULTITHREADING** – Intro to Python threading
- **11_WORKING_WITH_APIS** – Basic API interaction
- **12_dunder_name** – Understanding `__name__ == "__main__"`
- **EXERCISES** – Practice projects (calculator, quiz game, banking app, alarm clock, and more)

## ✅ Prerequisites

- Python 3.10+ (3.11+ recommended)
- A terminal (Linux/macOS/Windows PowerShell)
- Optional: VS Code

## ▶️ How to run

From the project root:

```bash
python main.py
```

Run any lesson file directly:

```bash
python 01_BASICS/01_variables.py
python 02_CONTROL_FLOW/05_for_loop.py
python 07_OBJECT_ORIENTED_PROGRAMMING/03_inheritance.py
```

Run exercise examples:

```bash
python EXERCISES/02_calculator.py
python EXERCISES/12_banking_program.py
```

## 🎯 Learning goal

The aim of this repository is to build a strong Python foundation that supports backend development concepts such as:

- clean code structure
- modular programming
- error handling
- file and API operations
- OOP principles used in real backend systems
